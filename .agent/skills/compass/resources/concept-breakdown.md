# COMPASS: Concept Breakdown Template

Use this template to bridge the gap between "knowing how" and "understanding why".

## 1. The Immediate Solution (The "Now")
Quick code snippet or command to solve the current blocker.

## 2. The Underlying Concept (The "Why")
- **Visual Analogy**: (e.g., "Think of a Closure as a backpack that a function carries around...")
- **Mental Model**: Explain the abstraction or design pattern used.

## 3. Practical Reinforcement (The "Challenge")
A small task for the user to try independently to verify mastery.
> "Now try to modify this logic to support [Case X] without changing the existing interface."
