# Template Skill

## Overview
This skill serves as a reference for the expected format of a `SKILL.md` file.

## Instructions
1.  **Modify Frontmatter**: Update the name and description.
2.  **Detail Workflow**: Break down the process into clear steps.
3.  **Add Tips**: Include common pitfalls and best practices.

## Best Practices
- Keep instructions concise and actionable.
- Use headers to organize content.
- Provide real-world examples.
