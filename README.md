# Software Inventario

Un sistema completo de gestión de inventario.

## Instalación Rápida (Windows)

¡Hemos simplificado la instalación para ti!

1.  **Descargar**: Clona este repositorio o descárgalo como ZIP.
2.  **Instalar**: Haz doble clic en el archivo `install.bat`.
    - Esto instalará automáticamente todas las dependencias necesarias para el Backend y el Frontend.
    - Asegúrate de tener Node.js instalado previamente.
3.  **Iniciar**: Una vez finalizada la instalación, haz doble clic en `start.bat`.
    - Esto abrirá dos ventanas (servidor y cliente) y lanzará la aplicación en tu navegador.

## Requisitos Previos

- [Node.js](https://nodejs.org/) (versión 16 o superior recomendada)
- Git (opcional, para clonar el repositorio)

## Estructura del Proyecto

- `backend/`: Servidor API (Node.js/Express)
- `frontend/`: Cliente Web (React/Vite)
